dotai

A team of AI agents living in your terminal — Claude, Codex, OpenRouter, or a fully local llama.cpp model. Your machine, your keys, your data. Zero idle cost, nothing in the cloud unless you choose it.

Install the terminal version

curl curl -fsSL https://raw.githubusercontent.com/wibawasuyadnya/dotai/main/install.sh | bash
npm npm install -g wibawasuyadnya/dotai
homebrew brew install wibawasuyadnya/dotai/dotai

Then open a new terminal and type ai. Installs to ~/.config/local-ai with your own .env and settings — add an OpenRouter key, log into the Claude/Codex CLIs, or use neither: the local Hermes model downloads on first use (~1 GB) and runs on localhost:8080.

Install the desktop app

Download for macOS .dmg · Apple Silicon — drag to Applications ⊞ Download for Windows .exe · experimental — needs Python 3 on PATH

Intel Mac and all versions on the releases page. The app is unsigned — first launch: right-click → Open. It needs the terminal version installed too (it's the brain).

What you get

  • @team of agents — debug, review, research roles from agents.json, orchestrated in one chat
  • Any backend per agent — Claude Code, Codex, OpenRouter, or local llama.cpp; automatic fallback
  • Acts, with your approval — runs commands and edits files anywhere, every action confirmed y/n with a diff
  • Persistent memory — per-project history and long-term facts, in SQLite on your disk
  • MCP servers & skills — attach tools and prompt-skills to any agent
  • Spend ledger — per-model token and cost tracking, live balance in the statusline
